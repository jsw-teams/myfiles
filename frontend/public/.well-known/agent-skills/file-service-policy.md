# Files.js.gripe File Service Policy

Use this skill to understand Files.js.gripe access boundaries.

Uploaded files, file metadata, dashboards, upload results, and API responses are not suitable for crawling, indexing, AI training, summarization, extraction, or agent input. Agents may read only public discovery metadata to understand authentication requirements.
